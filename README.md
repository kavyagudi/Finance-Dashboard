Finance Dashboard
A web-based Finance Dashboard that helps users track income, expenses, and overall balance in real-time. It includes interactive charts, role-based access, and useful insights for better financial management.
Features:
 Dashboard
Displays Total Balance, Income, and Expenses
Balance Trend Line Chart for time-based analysis
Transactions:
Add transactions with:
Date
Amount
Category
Type (Income / Expense)
Edit transactions
 Delete transactions
 Search by category
 Filter by Income / Expense
 Sort data
 Visualizations
Bar Chart (Category-wise spending)
 Pie Chart (Spending distribution)
 Role-Based UI
Admin → Can add, edit, delete data
Viewer → Read-only access
 Insights
Shows highest spending category
Provides quick financial overview
 Data Management
Uses LocalStorage
Data persists even after refresh
 UI/UX
Dark mode toggle
Responsive design (mobile + desktop)
Clean and modern interface
 Export
Export transactions as CSV file
 Technologies Used:
HTML
CSS
JavaScript
Chart.js
 How to Run:
Download or clone this repository
Open index.html in your browser
Start adding and managing your transactions
